Що таке HTML?

HTML (HyperText Markup Language) — це стандартизована мова розмітки документів, яка використовується для створення структури веб-сторінок. Вона не є мовою програмування, а слугує "каркасом", на який згодом накладаються стилі (CSS) та логіка (JS).

Навіщо він потрібен?

  • Структурування: Визначає заголовки, абзаци, списки та таблиці.
  • Зв'язок: Створює гіперпосилання для переходу між сторінками.
  • Мультимедіа: Дозволяє вбудовувати зображення, відео та аудіо.
  • SEO: Допомагає пошуковим роботам індексувати контент.

Основи стилізації CSS3

CSS (Cascading Style Sheets) — це мова, яка вдихає життя в HTML-каркас. Вона дозволяє точно керувати кольорами, шрифтами, відступами та анімаціями, створюючи сучасний візуальний досвід.

Каскадність: Ієрархія правил, що визначає пріоритет стилів.
Селектори: Способи вибору елементів для надання їм вигляду.
Box Model: Система відступів (margin, padding) та меж (border).
Flex & Grid: Сучасні інструменти для побудови складних макетів.
color: #00d4ff;

Змінює колір тексту елемента.

background: #1e1e1e;

Встановлює колір або зображення фону.

border-radius: 12px;

Закруглює кути блоку.

padding: 20px;

Створює внутрішні відступи від межі до контенту.

Попередній перегляд
Box Model 
.box {
  background: #00d4ff;
  border-radius: 15px;
  box-shadow: 0 5px 15px;
}
